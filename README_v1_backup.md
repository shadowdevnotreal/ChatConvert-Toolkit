**Features:**
- ğŸ¨ Drag & drop file upload
- ğŸ”„ Real-time conversion
- ğŸ“Š Interactive analytics
- ğŸ’¾ One-click download

### Python API

```python
from chatconvert import ConversionEngine
from chatconvert.analytics import AnalyticsEngine

# Convert any format to any format
engine = ConversionEngine()
result = engine.convert(
    input_file='discord_export.json',
    output_file='chat.pdf',
    output_format='pdf'
)

# Analyze conversations
analytics = AnalyticsEngine(use_ai=True, groq_api_key='your-key')
parser = engine._get_parser('discord_export.json')
conversation = parser.parse('discord_export.json')
insights = analytics.analyze(conversation)

print(f"Sentiment: {insights['sentiment']['overall_sentiment']}")
print(f"Topics: {insights['topics']['main_topics']}")
```

### REST API

```bash
# Start Flask server
python -m chatconvert.web.app

# Convert file
curl -X POST -F "file=@chat.csv" -F "format=html" \
  http://localhost:5000/api/convert --output result.html

# Analyze file
curl -X POST -F "file=@chat.csv" \
  http://localhost:5000/api/analyze
```

### CLI

```bash
# Interactive menu
python menu.py

# Direct conversion
chatconvert convert input.json --to pdf --output chat.pdf

# Batch processing
chatconvert batch *.csv --to html --output-dir converted/

# Analytics
chatconvert analyze chat.csv --ai --api-key your-key
```
